## 시스템 개요

### 핵심 개념

```
┌─────────────────────────────────────────────────────────────┐
│                    30분 = 1 블록                            │
├─────────────────────────────────────────────────────────────┤
│  :00-:04   │        :05-:29        │  :30-:34  │  :35-:59  │
│  게이트    │        몰입 시간       │  게이트   │  몰입 시간 │
│  OPEN      │        CLOSED         │  OPEN     │  CLOSED   │
│  (5분)     │        (25분)         │  (5분)    │  (25분)   │
└─────────────────────────────────────────────────────────────┘
```

### 주요 설정값

| 항목 | 값 | 설명 |
|------|-----|------|
| 시작 시간 | 05:00 | 하루의 첫 블록 |
| 총 블록 수 | 48개 | 24시간 ÷ 30분 |
| 게이트 시간 | 5분 | 세션 선택 가능 시간 |
| 블록 시간 | 30분 | 한 사이클 |
| 매칭 규칙 | 2-3명 | 4명이면 2+2 분리 |

### 운영자 실제 작업 요약

| 시점 | 작업 | 소요 시간 | 빈도 |
|------|------|----------|------|
| 전날 저녁 | 내일 시트 생성 버튼 클릭 | 30초 | 매일 1회 |
| 운영 중 | (자동) 시스템이 모든 것 처리 | - | 자동 |
| 긴급 시 | 강제 게이트 열기/닫기 | 10초 | 거의 없음 |
| 마무리 | 마스터 복귀 (선택) | 10초 | 선택사항 |

**결론: 하루 총 운영 시간 약 1분**

## 사용자 여정 상세

### 1단계: 최초 등록 (1회성)

```
┌─────────────────────────────────────────────────────────────┐
│ 📧 사용자: 운영자에게 참여 요청                              │
│                                                              │
│ "세션풀에 참여하고 싶어요!"                                  │
│ "이메일: user@example.com"                                   │
│ "인스타: @awesome_user"                                      │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ 👤 운영자: 사용자 등록                                       │
│                                                              │
│ [관리자 메뉴] > [➕ 사용자 등록]                             │
│ ├─ 이메일: user@example.com                                 │
│ ├─ 인스타그램: @awesome_user                                │
│ └─ 코호트: @각자 (기본)                                     │
│                                                              │
│ 시스템 동작:                                                 │
│ ├─ 시트에 새 행 추가: "⭐00 @awesome_user"                  │
│ ├─ Script Properties에 사용자 정보 저장                     │
│ └─ 드롭다운 유효성 검사 적용                                │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│ ✅ 등록 완료                                                 │
│                                                              │
│ 사용자 상태:                                                 │
│ ├─ streak: 0 (⭐ 이모지)                                    │
│ ├─ cohorts: ["@각자"]                                       │
│ └─ lastParticipation: null                                  │
└─────────────────────────────────────────────────────────────┘
```
